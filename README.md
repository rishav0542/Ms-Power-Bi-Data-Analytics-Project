# IKEA Customer Retention Analytics

![IKEA Logo](https://via.placeholder.com/800x200?text=IKEA+Customer+Retention+Analytics) <!-- Replace with actual image if available -->

## Overview
A Power BI dashboard dissecting IKEA's customer data for retention insights. Analyzes churn drivers, loyalty tiers, customer lifetime value (CLV), and store impacts to recommend data-backed strategies reducing churn by 10-15%.

Key elements:
- **Data**: Transactions, demographics, loyalty, stores (Super vs. Express).
- **Focus**: High-churn regions (London: 47%, Manchester: 50%), tier performance.

## Objectives & Tasks
1. **Data Prep (Power Query)**: Cleaned/merged tables; calculated churn (Inactive >180 days), CLV.
2. **Measures & Columns**: DAX for churn rate, avg spend, points earned/redeemed.
3. **Churn Analysis**: Overall 51.7%; High-income: 55%, Platinum tier: 56%.
4. **Loyalty Program**: 51% transactions with promos (+1% spend uplift); Silver earns most points, redeems least.
5. **Store/Region Insights**: Superstores: Higher spend; Manchester/London: 53-54% churn.
6. **CLV Segmentation**: Top 25% (> $384): 170 days inactive vs. 190 for low-CLV.
7. **Dashboards**: Churn by region/income/store; Purchase frequency; Promo impact.
